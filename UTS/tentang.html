Tentang Saya

Saya Adalah Achmad Herlambang Prasetyo. Saya sejak kecil suka membuat game. Game apa jika ditanya? Saya akan menjawab, saya lebih sering membuat game Role Playing top-down dengan grafik pixel. Kenapa pixel? Dari dulu saya sangat menyukai gambar pixel deari berbagai media. Baik itu game maupun art. Apa yang membuat saya suka? Mungkin karena usnik saja bentuknya dari yang berbentuk kotak kecil dapat dibuat menjadi sebuah grafik yang indah. Ini membuat saya berpikir. Kenapa saya tidak membuat game yang mempunyai grafik seperti ini saja? Dari situ saya mulai belajar tentang bagiamana caranya agar dapat membuat game yang mempunyai grafik pixel.

Saya mempunyai cita-cita. Suatu hari nanti saya ingin bekerja di perusahaan game terkenal seperti Bandai Namco atau Square Enix. Jika saya dapat bekerja di salah satu perusahaan tersebut, saya baru dapat tidur puas. Ini bukan hanya sebuah cita-cita saya belaka. Ini adalah ambisi saya agar dapat menjadi seorang Programmer sejati

Hobby saya? Hobby saya adalah menggambar. Jika dipikir-pikir. Hobby saya itu berbeda jauh ya dengan cita-cita saya. Mungkin iya, tapi saya tidak ingin hanya memiliki satu kemampuan. Ini disebabkan jika suatu hari nanti saya tidak mencapai galaxy, saya bisa jatuh diantara bintang-bintang.

Tetapi untuk mencapai cita-cita tersebut, saya tidak sendirian. Saya dibantu oleh Tuhan yang maha Esa, keluarga saya, dan juga sahabat-sahabat saya mulai dari saya SD sampai saya masuk ke Universitas ini. Saya sangat berterimakasih terhadap mereka yang sudah menjalin hubungan dengan saya.
